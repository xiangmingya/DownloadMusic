# 音楽 - Music Downloader

一个简洁的多平台音乐搜索、播放和下载工具。

## 功能特性

- **多平台支持**：网易云音乐、QQ音乐、酷我音乐
- **聚合搜索**：同时搜索所有平台并混合展示结果
- **在线播放**：支持在线试听，实时显示歌词
- **音质选择**：128k / 320k / FLAC / FLAC 24bit
- **歌单下载**：支持批量下载整个歌单
- **双模式搜索**：关键词搜索或 ID 直接查询

## 使用方法

1. 在浏览器中打开 `index.html`
2. 选择搜索模式（关键词/ID）和类型（单曲/歌单）
3. 输入搜索内容，点击"検索"按钮
4. 在结果列表中可以：
   - 点击"▶"按钮在线播放
   - 点击"下载"按钮下载歌曲

## API 接口

本项目使用 TuneHub 音乐解析服务：`https://music-dl.sayqz.com`

主要接口类型：
- `info` - 获取歌曲信息
- `url` - 获取播放/下载链接
- `pic` - 获取封面图片
- `lrc` - 获取歌词
- `search` - 平台搜索
- `aggregateSearch` - 聚合搜索
- `playlist` - 获取歌单

## 技术栈

- 纯前端实现（HTML + CSS + JavaScript）
- 无需后端服务器
- 使用 Fetch API 进行数据请求

## 注意事项

- 需要网络连接访问音乐 API 服务
- 下载功能会在新标签页中打开音频文件
- 服务状态会每分钟自动检查一次
